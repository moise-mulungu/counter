# Counter

This is a counter app that allows users to count number either by increasing, decreasing and resetting.

# How to run the app
``git clone``: git@github.com:moise-mulungu/counter.git

``cd counter``

``live server``

# Built With
 - JavaScript
 - HTML/CSS
